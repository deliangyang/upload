

这个项目是百度前端团队开发的一个框架，http://fex.baidu.com/webuploader/demo.html，

我将其移植过来，并做了一个测试的demo

#### 配置环境

```
composer install
```

#### 开启服务器
```
php -S localhost:80 -t 当前目录
```


#### demo 访问地址

```
http://localhost/test.html
```
